# openlayersViewer

openlayersViewer is a viewer map application.

## Running the prototype

* `git clone` this repo
* `cd openlayersViewer` and `npm install`
* run `npm start`
* open browser on http://localhost:1234/

## Built With

* [Openlayers 6.4.3](https://openlayers.org/)
